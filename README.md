# ViewIn3D

Wanna make a visualizer for 3D Objects? here a tool that is useful for you (in development)

## Tutorials:
### Make a window
```py
from ViewIn3D import visualizer

app = Visualizer("My visualizer", 1920, 1080)

app.run()
```
